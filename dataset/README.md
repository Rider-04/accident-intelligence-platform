## Dataset

This folder is intended to store the SQL dataset used in this project.
The full dataset used in this project is not included in this repository because it exceeds GitHub’s 100 MB file size limit.

### Download the dataset

Download the SQL dataset from Google Drive:

[Download Dataset](https://drive.google.com/file/d/1HiV25RHF6Eag-sT625eZI5u30nh7B8S5/view?usp=drive_link)

### Where to place it

After downloading, place the file inside the `dataset/` folder in the project root:

```text
accident-intelligence-platform/
│
├── dataset/
│   └── accident_dataset.sql
├── notebook/
├── README.md
└── requirements.txt
```

### Notes

* Make sure the file is placed exactly inside the `dataset/` folder.
* The notebooks/scripts in this project expect the dataset to be available locally.
* The dataset is hosted externally only because GitHub does not allow files larger than 100 MB in a standard repository.
